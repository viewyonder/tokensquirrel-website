# tokensquirrel-website

Marketing site for [tokensquirrel.com](https://tokensquirrel.com). Astro + Cloudflare Workers.

## Develop

```bash
npm install
npm run dev              # Astro dev server
npm run dev:wrangler     # Build + run under wrangler locally
```

## Deploy

```bash
npm run deploy           # astro build && wrangler deploy
```
